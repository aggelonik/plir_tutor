# ΑΕΠΠ Tutor Agent

Διαδραστικός βοηθός για το μάθημα **Ανάπτυξη Εφαρμογών σε Προγραμματιστικό Περιβάλλον (ΑΕΠΠ)** Γ' Λυκείου.

**Αγγελόπουλος Νικόλαος — Καθηγητής Πληροφορικής**

---

## Λειτουργίες

- 📋 **Ανάλυση λύσης** για κάθε θέμα Γ & Δ (δομές, μεταβλητές, λογική)
- 💻 **Κώδικας σε ΓΛΩΣΣΑ** πλήρης και σωστός συντακτικά
- ⚠️ **Συχνά λάθη** μαθητών
- 📎 **Upload PDF/εικόνας** θέματος πανελληνίων
- 🖨️ **Εξαγωγή PDF** σημειώσεων με κεφαλίδα καθηγητή
- 3 modes: Tutor / Λύση θέματος / Δημιουργία διαγωνίσματος

## Εγκατάσταση

### Βήμα 1 — Ανέβασε στο GitHub Pages

1. Δημιούργησε νέο repository στο [github.com](https://github.com) (π.χ. `aepp-tutor`)
2. Ανέβασε το αρχείο `index.html`
3. Πήγαινε στο repository → **Settings** → **Pages**
4. Source: **Deploy from a branch** → Branch: **main** → Folder: **/ (root)**
5. Κλικ **Save** — σε 1-2 λεπτά η εφαρμογή είναι στο:
   `https://ΧΡΗΣΤΗΣ.github.io/aepp-tutor/`

### Βήμα 2 — API Key

Κατά την πρώτη εκκίνηση θα σου ζητήσει το **Anthropic API Key**:
1. Πήγαινε στο [console.anthropic.com](https://console.anthropic.com)
2. **API Keys** → **Create Key**
3. Αντίγραψε το key (ξεκινά με `sk-ant-...`)
4. Επικόλλησε στο modal της εφαρμογής → **Αποθήκευση**

Το key αποθηκεύεται μόνο στον browser σου (localStorage) και δεν φεύγει πουθενά.

## Χρήση

- Επίλεξε mode (Tutor / Λύση / Δημιουργία)
- Γράψε ερώτηση ή επικόλλησε εκφώνηση
- Ή ανέβασε PDF/εικόνα θέματος
- Για εξαγωγή PDF: κλικ **Εξαγωγή PDF** → Print → Save as PDF

## Κόστος

Το Anthropic API χρεώνει ανά χρήση. Για εκπαιδευτική χρήση ΑΕΠΠ το κόστος είναι πολύ μικρό (μερικά cents/μήνα). Μπορείς να βάλεις spending limit από το console.
